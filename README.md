Priority Matrix Project

A simple and interactive Priority Matrix Web Application built using frontend technologies.
This project helps users organize their tasks based on priority and urgency using the Eisenhower Matrix method.

📌 Project Overview

The Priority Matrix is a task management interface that allows users to categorize tasks into four different sections:

Important & Urgent
Important but Not Urgent
Urgent but Not Important
Neither Urgent nor Important

This project focuses on creating a clean and user-friendly frontend interface for better productivity and task organization.

✨ Features
Add tasks to different priority sections
Simple and responsive UI

Interactive task management interface
